## 🎥 Filming Tips

- Record screen at 1920x1080
- Use a dark theme browser (matches the dark UI)
- Pre-fill some data before recording so the dashboard isn't empty
- Add captions/subtitles for accessibility
- Background music: soft instrumental, ~60 BPM
- Recommended tool: Loom, OBS, or QuickTime
